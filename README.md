# Projeto Site de Segurança Eletrônica

Este repositório contém o código-fonte para um site profissional de uma empresa de segurança eletrônica. O site é responsivo e projetado para fornecer informações sobre os serviços oferecidos, a empresa, opções de preços e um formulário de contato.

## Estrutura do Projeto

```
siteseg
├── index.html            # Página inicial do site
├── pages                 # Páginas adicionais
│   ├── services.html     # Serviços oferecidos
│   ├── about.html        # Informações sobre a empresa
│   ├── contact.html      # Formulário de contato
│   └── pricing.html      # Opções de preços
├── css                   # Folhas de estilo
│   ├── styles.css        # Estilos principais
│   └── responsive.css     # Estilos responsivos
├── js                    # Scripts JavaScript
│   ├── navigation.js     # Lógica de navegação
│   ├── form-validation.js # Validação do formulário de contato
│   └── interactions.js    # Interações adicionais
├── images                # Ativos de imagem
└── README.md             # Documentação do projeto
```

## Instalação

1. Clone o repositório:
   ```
   git clone <URL do repositório>
   ```
2. Navegue até o diretório do projeto:
   ```
   cd siteseg
   ```
3. Abra o arquivo `index.html` em um navegador para visualizar o site.
